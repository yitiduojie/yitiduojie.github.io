# Calculate the average of a column of data (求某列数据的平均值)

Author: Huanggang08

Problem： Assume that you get a trajectory named NO3-2.ener. Extract one column, eg., the 5th column, and to calculate the average and deviation of them.

NO3-2.ener

\# step Nr.            Time [fs                Kin. [a.u.]               Temp. [K]                 Pot. [a.u. ]                 Cons. Qty. [a.u.]                    Used Time [s]

6861                   3430.500000      0.017100803           300.0000000            -137.810608977    -137.784886346   360.690000000   

6862                      3431.00000    0.016742656        293.717014094             -137.810247862   - 137.784884019    85.5870000000

Solution:

step 1：

awk '{i++; s=s+$5; printf "%d %10.6f\n", $1, s/i}' NO3-2.ener > ave

Notes: the above command calculate the average of the 5th column in each step (record), therefore, by using it , we get the evolution of the average value of the 5th column (potential). The file named “ave” is a two-column file.

  1. $5 denotes the datum in 5th column.

  2. Printf: print formatted data to stdout. The format tags follow this prototype:

%[flags] [width] [.precision] [length] specifier

Eg., d means signed decimal integer (有符号的十进制整数)

Using this command, we can get the convergence process of the average .

step 2: plotting the figure.

Here, we find the average value is -137.804, and the deviation is 0.00392435.

Open the gnuplot, the command is:

gnuplot

after get into the gnuplot，type in the following commands:

plot ‘ave’ u 1: 2 w l  
f(x)=c
fit  f(x) ‘./NO3-2.ener’ u  1: 5  via  c
plot ‘NO3-2.ener’  u  1: 5 w  l  
replot  -137.804
rep -137.804 +0.00392435
rep -137.804 -0.00392435

Using the last three lines, we make three signs on the average value and the boundaries of the deviation.
