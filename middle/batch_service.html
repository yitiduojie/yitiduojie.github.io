# vmd教程

作者: 黄刚 日期：2020年1月8日

为了在VMD中显示周期性格子(PBC box):

vmd> molinfo top set {a b c} {40 19.7 19.7}
vmd> pbc box -on

VMD中设置周期性边界显示,于Tk condole中设置

pbc set {a b c 90 90 90}
