# CP2K

作者: Huang Gang

日期：2019年3月27日

在原子层次上对物质做计算机模拟成为了物理，化学，生命以及材料科学领域中一个非常有力的工具。在这些领域中，模拟的结果可以帮助人们解释实验上的测量结果，也可以用来预测材料的性质（比如预测晶体的结构[Sholl, David S. Chapter 2.3]）.

对原子模拟而言，CP2K计算机程序是一个非常强大的工具。 它特别适合于大的凝聚相系统的模拟(模拟的原子个数可达数百个)。

# How to generate an input file for CP2K run

Nicolai Ree: "We have recently published a simple and interactive GUI that allows you to create CP2K input files. The program is called CP2K Editor, and it provides a user-friendly interface for both new and experienced users. The program is freely available on Github (https://github.com/avishart/CP2K_Editor) and you can find information about the program in the following paper: Vishart, A.L., Ree, N. & Mikkelsen, K.V. J Mol Model (2019) 25: 115. https://doi.org/10.1007/s00894-019-3987-6 "
