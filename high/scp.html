# scp命令

作者: Dl

日期：2019年10月16日

# scp基础

Linux scp命令的目标：实现在Linux之间复制文件和目录。scp 是secure copy的缩写。

本质：基于ssh登录；加密的远程文件复制。

语法：

​ scp [可选参数] file_source file_target

参数：

-P port ：传到指定端口时用。port 用端口号代替。

-r : 递归复制整个目录。

-6 ： 强制scp命令只使用IPv6寻址。

# 实际操作

如果远程服务器为scp命令设置了指定的端口，我们需用-P参数来设置命令的端口号。例如，把文件libint-2.6.0.tar.gz复制到dylan@49.232.172.254的主文件夹，可用如下命令

​

​ $ scp -P 6000 /home/huang/libint-2.6.0.tar.gz dylan@49.232.172.254:~
