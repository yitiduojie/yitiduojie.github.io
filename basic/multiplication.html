# 乘法规则(3)

作者: Dylan Huang 日期：2019年4月17日

​我们现在学习整数的乘法。通过一个例子，我们可以看到在计算两个数相乘的过程中我们用到了一些计算==规则==[1]。重要的事，就是理解并记住这些规则。 让我们来计算一下两个数相乘.

计算$37\times254$.

解：$37\times 254 = (30 + 7)\times 254$

$= 30\times 254 + 7 \times 254$　（加法分配律）

$= 30\times (200 + 50 + 4) + 7\times (200 + 50 + 4)$

$= 30\times 200 + 30\times 50 +30\times 4 + 7\times 200 + 7\times 50 + 7\times 4$ （加法分配律）

$= 6000 + 1500 + 120 + 1400 + 350 + 28$

$= 7500 +1520 + 378$

$= 9020 +378$

$= 9398$

乘法的计算就完成了。然而$30\times200=6000$虽然看起来很显然，但它是怎么得来的呢? 现在我来证明: $30\times 200 = 6000$

证: 30的定义为: $3\times 10$,

200的定义为：$2\times (10\times 10)$

所以，$30\times 200 = (3\times 10) \times 2\times (10\times 10)$

$= (3\times 2) \times(10\times 10 \times 10)$ (乘法规则3:==交换律==: 对于任意两个数$a,b$,有$ab = ba$)

$= 6 \times 1000$

$= 6000$.

总之，在计算乘法的时候，我们需要的是：(1)使用两个规则－－加法分配律和乘法交换律；(2)记住简单的数相乘之结果(10以内的数之乘法表)；(3)会做加法。

[1]高尔斯,牛津通识读本:数学.
